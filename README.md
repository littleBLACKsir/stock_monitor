# AI算力产业链股票分析系统

基于文件驱动的AI股票分析工作流，专注于AI算力产业链股票的超跌买入机会发现。

## 项目结构

```
ai-stock-analyzer/
├── AGENTS.md              # AI分析师工作指南
├── config/
│   ├── stocks.yaml        # 股票池配置（16只股票+ETF推荐）
│   └── thresholds.yaml    # 技术指标阈值配置
├── templates/
│   └── analysis-template.md  # 分析报告模板
└── reports/               # 生成的分析报告目录
```

## 使用方法

1. 让AI读取 `AGENTS.md` 了解工作流程
2. AI根据 `config/stocks.yaml` 获取股票池
3. AI搜索股票技术指标数据
4. 根据 `config/thresholds.yaml` 计算超跌评分
5. 生成分析报告保存到 `reports/` 目录

## 投资策略

- **核心策略**：超跌买入，价值投资
- **分析周期**：每周四分析一次
- **评分标准**：超跌评分≥50分考虑买入
